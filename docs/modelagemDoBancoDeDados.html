Introdução

Este artefato possui como finalidade a documentação da modelagem do banco de dados, que consiste na criação do modelo relacional no software brModelo, versão 3.2, do projeto Conecta-Ensina, desenvolvido nas disciplinas de Métodos de Desenvolvimento de Software e Engenharia de Produto de Software no primeiro semestre do ano de 2020, que possui como cliente a empresa Conecta-Ensina, representado na equipe pelas figuras do Fábio e da Carol.

Modelagem do banco de dados

Modelo Entidade-Relacionamento (MER)

Entidades

  • USER (ID, firstName, lastName, email, password, cellphone, role)
  • STUDENT (idUser, birthDate, cpfParent, address(CEP, street, number, complement), institution, serie)
  • SUBJECT (ID, graduation, name)
  • TEACHER (idUser, cpf, address(CEP, street, number, complement), photo, description, video, institution, graduation, graduationArea, degree, subject [], bank, agency, account, status)
  • CLASS (ID, teacher, student, serie, subject, dtClass, duration, address(CEP, street, number), status, timer)
  • RATING_TEACHER (class, stars, rating)
  • RATING_STUDENT (class, rating)
  • REPORT (ID, teacher, report)
  • FAQ (ID, question, answer)

Cardinalidades

  • USER é pode ser um STUDENT, STUDENT é um USER.
    USER 0:1 STUDENT

  • USER é pode ser um TEACHER, TEACHER é um USER.
    USER 0:1 TEACHER

  • TEACHER pode dar várias SUBJECTS, SUBJECT pode ser dada por vários TEACHERS.
    TEACHER N:N SUBJECT

  • TEACHER pode dar várias CLASSES, CLASS é dada por um TEACHER.
    TEACHER 1:N CLASS

  • TEACHER pode ter vários REPORTS, REPORT pertence a um TEACHER.
    TEACHER 1:N REPORT

  • STUDENT pode receber várias CLASSES, CLASS é dada a um STUDENT.
    STUDENT 1:N CLASS

  • STUDENT pode fazer vários REPORTS, REPORTé feito por um STUDENT.
    STUDENT 1:N REPORT

  • CLASS é de uma SUBJECT, SUBJECT pode ter várias CLASSES.
    CLASS N:1 SUBJECT

  • CLASS pode ter um RATINGTEACHER, RATINGTEACHER é a uma CLASS.
    CLASS 0:1 RATINGTEACHER

  • CLASS pode ter um RATINGSTUDENT, RATINGSTUDENT é a uma CLASS.
    CLASS 0:1 RATINGSTUDENT
